# Human-in-the-loop

## Decisioni che richiedono approvazione
- Nuovi accessi o credenziali.
- Nuovi provider AI o servizi esterni.
- MCP/plugin.
- Trattamento dati personali.
- Deploy o modifiche a produzione.
- Modifiche a sicurezza, compliance, logging, retention.
- Rimozione massiva di codice o documentazione.

## Formato escalation
```text
Decision needed:
- Contesto:
- Opzioni:
- Raccomandazione agente:
- Rischi:
- Impatto compliance/security:
- Decisione richiesta:
```
