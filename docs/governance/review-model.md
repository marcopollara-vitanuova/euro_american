# Review Model

## Review a tre livelli
1. Self-review agente: controlli automatici e riepilogo rischi.
2. Technical review: qualità, test, architettura.
3. Governance review: sicurezza, privacy, compliance, accessi.

## Criteri di approvazione
- Scope rispettato.
- Nessun segreto/dato personale improprio.
- SDD/ADR aggiornati.
- Test/check eseguiti.
- Rischi residui accettati o mitigati.
