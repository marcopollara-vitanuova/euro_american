# Cursor configuration - HARNESS

Questa cartella inizializza il repository per l'uso con Cursor.

## File principali

- `rules.md`: indice umano delle regole operative.
- `prompts.md`: indice dei prompt ufficiali del repository.
- `mcp.json`: configurazione MCP vuota per default, coerente con policy A0/A1.
- `rules/*.mdc`: regole applicabili da Cursor Agent.

## Fonte di verita'

Le regole derivano da:

- `AGENT.md`
- `PROJECT_CONTEXT.md`
- `docs/specs/SDD.md`
- `docs/sdd/02-architecture-decision-records.md`
- `docs/sdd/03-data-access-and-mcp-policy.md`
- `docs/sdd/04-security-compliance-controls.md`
- `docs/sdd/05-observability-and-feedback.md`
- `docs/governance/*.md`
